Kamae（構え）は、サーバーサイドのドメインコードを堅牢に設計・レビューするためのスタンスとガイド集です。

## ドキュメント

- [Rust](/docs/kamae/rust/) — 型で無効な状態を表現し、`Result` と境界 DTO で守る Rust 向けガイド
- [Python](/docs/kamae/python/) — Pydantic v2 の判別共用体と純粋な状態遷移関数を軸にした Python 3.12+ 向けガイド

## ソースリポジトリ

- [kamae-rs](https://github.com/manji-0/kamae-rs)
- [kamae-py](https://github.com/manji-0/kamae-py)

各リポジトリには Codex / Claude 向けスキルパッケージも同梱されています。このサイトのドキュメントは、ブログ読者向けに日本語で整理した版です。
