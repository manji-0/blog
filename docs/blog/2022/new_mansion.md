# SRE、マンションを買う

*システムの信頼性は論理的に考えて定義するが、新築マンションを買う決断は勘で決めた*

## 数時間の判断
2021年の3月頃、インスタで流れてきた新築マンションの広告が目に入った。話を聞いてみてもいいな、と思った。

というのも、当時の僕は住環境...というより公私が切り分けられていない環境で24時間を過ごすことに耐えがたいストレスを感じていたからだ。

コロナが流行り出し在宅勤務が始まったのが2020年の春くらいだったかと思う。丁度在宅勤務を始めて1年くらいが経っていた。

仕事机の横にマットレスと布団が敷かれ、仕事椅子の後ろには筋トレ用のベンチがあり、あたりは取り込んだ洗濯物や本で足の踏み場もないという過酷な住環境に嫌気が差すには1年という時間はむしろ長い、良く耐えてきたというべきだろう。

その日のうちに説明会のアポを取り、細かいスペックを聞いた。いい部屋もまだ空きがある。僕の直感は買えと言っている。

そこから2~3時間ほどかけて「直感を検証するための確認」を丁寧に済ませた後、翌日に弟とモデルルームに行ってイメージを固め、その日のうちに売買契約のサインを書いた。

当時から1年と4ヶ月ほど経ち、先週からようやく新居に住み始めた。すこぶる快適である。

以下は当時考えたこと・感じたことであったり、実際どのくらい費用がかかったかについての記録である。

## 新築マンションを買うにあたって考えたこと
僕が買った部屋は「首都圏、駅近、狭めの3LDK」というよくあるものであり、ザックリと6300万円ほどの値がついていた。

安くはない買い物なので、判断の前に多少は費用対効果を検証する時間をとった方が得だ。

* 断熱・遮音性能は? → スラブ厚、壁厚(角部屋だ)、界壁厚はすべて200~250mmが確保される設計。窓はT4サッシ。第三種換気。
* 駅へのアクセスは? → 僕の通ってるジム、病院、歯医者などの最寄り駅へ徒歩10分圏内、物件からの最寄り駅は細い道の向かいにあり、徒歩0分。
* 間取りは? → 3LDK。以下のようにしてピッタリ配分できる。
    * 1部屋目：弟の部屋
    * 2部屋目：僕の寝室
    * 3部屋目：僕の仕事部屋
* 収納は? → 玄関、リビング、洗面所、各部屋に十分なスペースがあった。男2人分の生活用品は余裕で収まる。
* 設備は? → 広い台所、広い洗濯機置き場、(比較的)広い風呂がついており、ビルトイン食洗機もオプションで入れられる。
* 地盤は? → 川沿いなので少し不安だが、基礎杭の長さはかなり長めのようだし、住むのは中層階。いざという時の避難も難しくない立地。

...というようなことを営業と一緒に図面や地図を見て確認した上で、「恐らく同様の部屋を賃貸で借りるくらいなら買った方がかなり得だろう」という判断が間違っていないと評価した。

(冒頭で述べたがこのあたりのことはパンフを一読したあたりで直感が統合的に判断してくれている。具体的な評価をしたのはあくまでその確認である)

モデルルームでは内装の質感やコンセントの規格、カスタム可能な箇所を確認した。懸念が残ることは無かった。モデルルームはとても綺麗に演出されており、とてもテンションが上がったことを覚えている。

## 設計変更
売買契約をしたら次は設計変更のfixをしないといけない。1時間くらい悩んで決めた。

基本的な構造は一切変更していない。食洗機を付けることは契約書を書くときから決めていた。では何に悩んでいたのかというと、以下の3つだ。

* コンセントの位置
* 内装の配色
* ドアノブ

### コンセントの位置
今どきの狭めの2~3LDKには当たり前なのかもしれないが、少なくとも僕が買った部屋はマルチエアコン限定だった。そして図面上では何故か室内のエアコン室内機の近くにそれぞれコンセントが配置されていた。

...詳しい方はピンと来たと思うが、これは無駄である。現在販売されているマルチエアコンはほとんどが室外機からの給電で、室内機からは電源端子など延びてはいない。

念のため確認してみたが、ベランダにはきちんとエアコン専用の200Vコンセントが設置されるらしい。つまり純粋に無駄なコンセントだ。

設計変更について、コンセントの位置移動は無料とのことだったので、便利そうな位置に動かすことにした。仕事机の横、ベッドサイド、WICの中など。この配置を決めるのに10分ほどかかった。
